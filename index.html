About

Break-All is a solid game which can be playing in the desktop browser as also as mobile. It consists like Angry Bird to break some objets in the area.

The screens give a realistic gameplay and support mouse or touch. It can be play with background sound who give more truly state of the scene. The player can share his score online and see for all others.

This game was developed by Jerome Dh on its hobby time and it's free and open source.
For user player, the game is playing online by connecting on his address, this allow user to play without any dependancy.
For user Developper, the source code is shared on Github. They can clone and modify on their own convenience.
